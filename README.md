# CloudMusic

#### 介绍
Vue版仿网易云音乐，

[http://catbk.cn:3000【项目预览】]: 



#### 软件架构
后端数据来源于网易云音乐接口

##### 使用技术

- vue2.6.11
- vue-router3.1.5
- axios0.19.2
- element-ui2.13.0




#### 使用说明

1.  npm install
2.  npm run serve
